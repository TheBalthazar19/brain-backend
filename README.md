# Memory Assistant Backend

AI-powered memory assistant backend built with Node.js, Hono, TypeScript, PostgreSQL, and Pinecone for semantic search.

## Features

- 🔐 **Authentication**: JWT-based auth with BetterAuth
- 📝 **Memory Management**: Full CRUD operations for memories
- 🔍 **Semantic Search**: RAG with Mistral AI and Pinecone
- 🛡️ **Security**: Rate limiting, CORS, input validation
- 🐳 **Docker**: Containerized deployment
- ☁️ **Cloud Ready**: Azure Container Apps deployment
